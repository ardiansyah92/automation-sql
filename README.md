# Automation SQL with Python Code

Remember guid for this script automation sql you must have table booking and table price list.



Install Pyhton 

    brew install python

Check Python 

    python3 --version
    pip3 --version

Install Pytest

    pip install pytest

Check Pytest

    pytest --version

Install PyMysql

    pip install PyMySQL

Check PyMysql

    python3 -c "import pymysql; print('OK', pymysql.__version__)"

Clone repo Automation SQL

    https://github.com/ardiansyah92/automation-sql.git

Running code

    pytest test_booking_validation.py

If you have error sqlalchemy error not detected
Choose Select Interpreter

    Python: Select Interpreter

Choose 

    Enter interpreter path

Input

    /Library/Frameworks/Python.framework/Versions/3.12/bin/python3

Running again code     

    pytest test_booking_validation.py

please don't consent with the FAILED result but focus from result query SQL
